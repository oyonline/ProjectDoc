# 流程与文档管理系统

企业内部流程与制度文件管理系统的高保真可交互 Web 原型。

## 功能特性

- **文档管理** — 企业制度/流程文件的创建、编辑、搜索与分类筛选
- **版本管理** — 完整版本历史、版本对比（diff 视图）
- **流程节点图** — 可交互 SVG 流程图，点击节点查看详情（负责人、SLA、说明）
- **发布与生效** — 审核 → 发布 → 生效完整工作流
- **权限/可见范围** — 四级角色体系 + 文档级可见范围配置
- **变更留痕** — 完整操作审计日志
- **阅读确认** — 跟踪员工阅读确认进度，支持催办提醒

## 技术栈

- React 18
- Vite
- 纯 CSS-in-JS（无额外 UI 框架依赖）

## 快速开始

```bash
npm install
npm run dev
```

浏览器打开 `http://localhost:3000`

## 构建

```bash
npm run build
```

## 截图

系统包含六大模块：工作台、文档管理、流程管理、变更日志、阅读确认、权限设置。
